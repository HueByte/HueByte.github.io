Quick command guide

The command syntax: prefix CommandName arguments
Arguments are divided by spaces, that means if you want to enter the sentence to command you need to put it in quotes
Example: $DailyRun "10;00 pm" "Some party"

In Some commands which intentionally take sentence you can skip quotes because they're taking only one argument
Example: $UrbanDefine blue car

When commands require user as argument you should use @user mention
Example: $GetAvatarUrl @H.u.e.
